# Email Polisher (Ansiversa Mini-App)

Premium drafting workspace for refining professional emails quickly.

## V1 capabilities

- Structured email basics (subject, recipient, sender, context)
- Three independent variants (Draft A, Draft B, Draft C)
- Tone selector and refinement flags
- Live preview of the active draft
- Copy current draft / copy all variants
- Local draft persistence with reset flow

## Commands

```bash
npm install
npm run typecheck
npm run build
```
